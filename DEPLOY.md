# Deployment Guide

Your project is built and ready in the `dist` folder! 🎉

## 1. Quickest Method: Netlify Drop

1.  Go to [app.netlify.com/drop](https://app.netlify.com/drop).
2.  Open your project folder in File Explorer:
    `c:\Users\Mustafa Yağız ÜNAL\.gemini\antigravity\scratch\paspartu`
3.  Drag and drop the **`dist`** folder onto the Netlify page.
4.  Your site will be live in seconds! You can then change the domain name in Netlify settings.

## 2. Professional Method: Vercel

1.  Push this project to a GitHub repository.
2.  Go to [vercel.com](https://vercel.com) and log in.
3.  Click "Add New..." -> "Project".
4.  Select your GitHub repository.
5.  Click "Deploy". Vercel will handle everything automatically.

## 3. Command Line (Advanced)
If you have Vercel installed locally:
`vercel deploy`
